Current Ruleset

This texture encodes the update rules of each cell:

-   Each cell has access to the sum of its neighbours color values, and compares that value against the sum of values given by this texture.

-   The blue value is matched if it is below the cells neighbour sum value. This is more effecient when checking that neighbouring cells contain at least one value without having to check all the possible permutations.

-   Neighbour sum values that leave the state unchanged can be ommitted from this texture, and the current value will be automatically copied to the updated state.
